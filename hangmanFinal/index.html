This program works by randomly selecting a four letter word from a pre-made array. Each word to guess does not have any repeating letters, so each four letters will be unique. Unfortunately, I was having issues figuring out how to make the program only accept the letters if they are guessed in the right order, so in order for it to work properly, the letters have to be guessed in the correct order, if the word contains the letter. If you are trying to find the word you need to guess, you can check the console, I have logged the word in the console. Also, there is in-depth detail of how my program works if you check the comments on the app.js file.
